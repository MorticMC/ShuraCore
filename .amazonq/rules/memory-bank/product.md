# ShuraCore - Product Overview

## Project Purpose
ShuraCore is a comprehensive Minecraft PvP practice and duels plugin designed for competitive PvP servers. It provides a complete infrastructure for ranked matches, practice modes, party systems, and player progression tracking.

## Value Proposition
- **Complete PvP Solution**: All-in-one plugin for competitive Minecraft PvP servers
- **Ranked System**: Dual queue system (cracked/premium) with tierlist rankings
- **Flexible Practice**: FFA practice modes and custom kit configurations
- **Social Features**: Party system for team-based gameplay
- **Professional Infrastructure**: Database-backed profiles, match logging, and statistics

## Key Features

### Core Gameplay
- **Duel System**: Player-to-player challenge system with kit selection
- **Ranked Queues**: Separate cracked and premium ranked matchmaking
- **Practice Modes**: Free-for-all practice arenas for skill development
- **Match Management**: Complete match lifecycle with rounds, spectating, and snapshots
- **Kit System**: Customizable kits with editor and validation rules

### Player Systems
- **Profile Management**: Persistent player statistics and progression
- **Tierlist Rankings**: Competitive ranking system with leaderboards
- **Party System**: Create and manage teams for group matches
- **Settings**: Customizable player preferences

### Arena Management
- **Dynamic Arenas**: Copy-based arena system for concurrent matches
- **Arena Editor**: In-game tools for creating and configuring arenas
- **Arena Validation**: Ensures arenas meet requirements before use
- **Auto-Reset**: Automatic arena restoration after matches

### Administrative Tools
- **GUI Editor**: Visual editor for customizing in-game menus
- **Maintenance Mode**: Server maintenance with whitelist bypass
- **Whitelist Management**: Control access during maintenance
- **Comprehensive Commands**: Full suite of utility and moderation commands

### Technical Features
- **Database Integration**: SQLite with HikariCP connection pooling
- **Scoreboard System**: Dynamic scoreboards with FastBoard
- **Permission Integration**: LuckPerms integration for rank-based features
- **Message System**: Configurable messaging with color support
- **Sound System**: Configurable sound effects for events

## Target Users

### Server Owners
- Looking for a complete PvP practice solution
- Need ranked competitive gameplay
- Want customizable kits and arenas
- Require player progression tracking

### Players
- Competitive PvP enthusiasts
- Practice mode seekers
- Team-based gameplay participants
- Ranked progression trackers

## Use Cases

1. **Competitive PvP Server**: Full-featured ranked duels with tierlist system
2. **Practice Server**: FFA practice modes for skill development
3. **Community Server**: Party system for friend-based gameplay
4. **Tournament Platform**: Match logging and spectator support
5. **Hybrid Server**: Combination of ranked and casual gameplay modes
